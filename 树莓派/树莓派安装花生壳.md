# 从花生壳官网下载安装包
```
https://hsk.oray.com/download/
```

# 安装软件
```
sudo dpkg -i phtunnel_5_0_rapi_armhf.deb
```

# 启动花生壳
```
sudo phddns start
```

# 设置花生壳开机启动
```
sudo phddns enable
```

# 设备管理
地址：http://b.oray.com   
将安装成功的花生壳SN添加到网页端设备管理，并激活设备
